# AmberTest

校园二手微信小程序 Demo

Lisence MIT

### 已有功能

- 按照类目发布、查看商品；
- 按照校区为用户、商品进行分类；
- 用户可以重设自己的地区信息；
- 商品下可以自由发表评论。
